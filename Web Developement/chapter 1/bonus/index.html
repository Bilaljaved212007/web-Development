HTML & CSS

HTML Tags

Headings (<h1> - <h6>)

Headings help organize your webpage. <h1> is the biggest, <h6> is the smallest.

<h1>This is a big heading</h1>
<h2>This is a smaller heading</h2>

Paragraph (<p>)

Use this tag to write regular text.

<p>This is a simple paragraph of text.</p>

Links (<a>)

Want to create a clickable link? Use this tag!

<a href="https://example.com">Visit Example</a>

Images (<img>)

Use this tag to display an image on your webpage.

<img src="image.jpg" alt="A description of the image">

Lists (<ul> and <ol>)

Want to create a list? Use <ul> for bullet points and <ol> for numbered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Tables (<table>)

Tables help organize data in rows and columns.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
    </tr>
</table>

Forms (<form>)

Forms allow users to input data.

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

Div (<div>)

The <div> tag is used to group elements together for styling.

<div class="container">
    <p>This is inside a div</p>
</div>

Span (<span>)

The <span> tag is used to style part of a text.

<p>This is a <span style="color:red;">red</span> word.</p>

CSS Properties

Text Color (color)

Change the color of your text.

p {
    color: blue;
}

Background Color (background-color)

Change the background color of your webpage or elements.

body {
    background-color: lightgray;
}

Font Size (font-size)

Adjust how big or small your text appears.

h1 {
    font-size: 32px;
}

Spacing Around Elements (margin)

Margin creates space outside an element to separate it from others.

div {
    margin: 20px;
}

Padding (padding)

Padding adds space inside an element.

div {
    padding: 10px;
}

Border (border)

Adds a border around an element.

div {
    border: 2px solid black;
}

Text Alignment (text-align)

Aligns text inside an element.

p {
    text-align: center;
}

Width and Height

Defines the size of an element.

div {
    width: 200px;
    height: 100px;
}
